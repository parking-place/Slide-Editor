# v0.9.0
Date: 2026-04-08

## Added

- Added a dedicated Project Manager modal for creating, opening, and saving projects without browser `prompt` dialogs.
- Added project rename and delete actions to the Project Manager, including automatic fallback to the next available project after deletion.
- Added drag-and-drop image upload support to the slide create and edit forms while keeping the existing file picker flow.

## Changed

- Changed the header project controls from separate `New`, `Open`, and `Save As` buttons to a single `Project` button that opens the Project Manager.
- Changed the project save/load workflow to use the same modal-driven UI pattern as the theme manager for a more consistent editing experience.

## Fixed

- Fixed the project deletion flow so the currently open project is replaced automatically after deletion instead of leaving the editor in an invalid state.
