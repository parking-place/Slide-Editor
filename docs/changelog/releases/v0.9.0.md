# v0.9.0
Date: 2026-04-08

## Added

- 브라우저 `prompt` 없이 프로젝트를 만들고, 열고, 다른 이름으로 저장할 수 있는 전용 `Project Manager` 모달을 추가했습니다.
- 프로젝트 이름 변경과 삭제 기능을 `Project Manager`에 추가했고, 현재 프로젝트 삭제 시 다음 프로젝트로 자동 전환되도록 구성했습니다.
- 슬라이드 생성/수정 폼의 이미지 업로드 영역에 드래그앤드롭 업로드를 추가했습니다.

## Changed

- 헤더의 `New`, `Open`, `Save As` 버튼을 하나의 `Project` 버튼으로 통합하고, 클릭 시 `Project Manager`를 열도록 변경했습니다.
- 프로젝트 저장/불러오기 흐름을 테마 모달과 유사한 고정형 관리 UI로 정리해 편집 경험을 일관되게 맞췄습니다.

## Fixed

- 현재 프로젝트 삭제 후 에디터가 비정상 상태로 남지 않도록, 삭제 직후 자동으로 대체 프로젝트를 로드하도록 수정했습니다.
