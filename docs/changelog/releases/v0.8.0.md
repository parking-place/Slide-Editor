# v0.8.0
Date: 2026-04-08

## Added

- 프로젝트 전용 저장소 API를 추가해 `data/projects/<projectId>/` 구조, 프로젝트 목록 조회, 생성, 삭제, 이미지 제공 흐름을 도입했습니다.
- `New`, `Open`, `Save As` 중심의 프로젝트 관리 UI를 추가했습니다.
- 루트 `version.json` 파일을 도입해 에디터 좌상단 버전 표시의 단일 기준을 만들었습니다.

## Changed

- 초기 로드와 저장 흐름을 현재 프로젝트 및 `app_state` 중심으로 전환했습니다.
- 이미지 저장 경로를 프로젝트별 `/api/projects/:projectId/images/:fileName` 흐름으로 분리했습니다.
- 기존 `slide_data.json`과 `data/image_data`를 `default` 프로젝트로 옮기는 자동 마이그레이션 준비 로직을 추가했습니다.
- 릴리즈 절차와 Docker 정리 규칙을 문서화했습니다.
