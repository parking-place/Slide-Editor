# v0.11.0
Date: 2026-04-10

## Added

- `New Project` 다이얼로그와 glass 테마 편집 UI를 확장해, 프로젝트 생성/임포트와 `Glass Surface` 설정을 한 흐름 안에서 다룰 수 있게 했습니다.
- `glass.noiseOpacity`와 `secondaryAccent`를 테마 스키마에 추가해 배경 질감과 보조 강조색까지 테마별로 저장할 수 있게 했습니다.
- 프로젝트 메타데이터에 `Last Saved`를 표시하고, 저장 시각을 `lastSavedAt`으로 기록하도록 연결했습니다.

## Changed

- 기본 테마를 `hpe_default`에서 `nvidia_light`로 변경했습니다.
- 에디터 상단 `Guide` 버튼을 `Viewer`로 바꾸고, HTML 가이드를 에디터 기반의 읽기 전용 `Slide Viewer` 셸로 재구성했습니다.
- `Viewer` 미리보기와 `HTML` 다운로드가 같은 최신 생성 경로를 공유하도록 정리했습니다.
- 단일 HTML 다운로드는 이미지/CSS/JS를 한 파일에 포함하고, 외부 폰트 링크는 유지하되 오프라인에서는 시스템 폰트로 자연스럽게 fallback 되도록 정리했습니다.
- glass 기본값, 배경 그리드/노이즈, 스크롤바, 헤더, Navigator, 모달, 프로젝트 매니저, 새 슬라이드 작성 UI를 전반적으로 다듬어 에디터와 Viewer의 시각 언어를 맞췄습니다.
- 프로젝트 매니저를 `Current Project` / `Selected Project` 구조로 재편하고, 복제 규칙을 `_copy` 형식으로 통일했습니다.

## Fixed

- 라이트 모드에서 `Glass Surface`와 색상 설정이 샘플뿐 아니라 실제 editor surfaces에도 반영되도록 복구했습니다.
- 같은 소제목이 여러 페이지에 걸쳐 있을 때 Navigator 활성 표시가 끊기던 문제를 수정했습니다.
- Viewer 배경 레이어를 editor와 같은 고정 방식으로 바꿔 긴 문서에서 그라디언트가 비정상적으로 늘어나지 않도록 했습니다.
- 구버전 JSON import/backfill 시 이미 WebP인 이미지는 변환을 건너뛰고, 실제 변환 작업이 없으면 완료 알림을 띄우지 않도록 정리했습니다.
- 코드 블록, 코드 색상, 이미지 교체/정리, 닫기 버튼 정렬, 한글 경고 문구 등 최근 UI/출력 흐름의 세부 버그를 함께 수정했습니다.
