# Unreleased

다음 릴리즈에 포함할 변경 사항을 여기에 기록합니다.

## Added

- 프로젝트 목록 헤더의 `+` 버튼으로 여는 `New Project` 다이얼로그를 추가했습니다.
- `New Project` 다이얼로그에서 새 프로젝트 생성과 JSON 데이터 Import를 분리해 시작할 수 있도록 정리했습니다.
- 글라스모피즘 테마 확장을 위한 `glass` 테마 스키마와 CSS 토큰 기본 구조를 추가했습니다.

## Changed

- 헤더의 프로젝트명 표시를 우측 버튼 묶음에서 분리하고, 중앙 glass 정보 패널로 재배치했습니다.
- 라이트모드 `Save` 버튼은 다른 헤더 버튼과 같은 glass 계열로 맞추고, primary action은 강조색 tint로만 구분하도록 조정했습니다.
- 다크모드 glass shadow와 highlight를 추가로 낮춰 전체 surface가 더 평평하게 보이도록 정리했습니다.
- 테마 목록 선택/호버, 테마 적용 버튼, 프로젝트 매니저 현재 뱃지와 주요 액션, 네비게이터 활성 텍스트 등 일부 녹색 하드코딩을 `accent`/`secondaryAccent` 기반으로 치환했습니다.

- 프로젝트 매니저 2차 UI 개편을 반영해 메인 액션을 `Current Project`와 `Selected Project` 상태에 맞게 다시 배치했습니다.
- 현재 프로젝트를 선택했을 때는 `Save`, `Copy`, `Export`만 보이도록 정리했습니다.
- 다른 프로젝트를 선택했을 때는 `Open`, `Copy`만 보이도록 정리했습니다.
- 기존 `Save As` 흐름은 `Copy` 동작으로 대체했고, 기본 복제 이름 규칙을 `_copy` 형식으로 맞췄습니다.
- `New`, `Import`, `Backup JSON`이 섞여 있던 하단 툴 박스를 제거하고, 역할에 맞는 별도 UI로 이동했습니다.
- `Current Project`와 `Selected Project` 카드의 메타 정보를 `ID`, `page`, `Saved Version`, `Last Saved` 4개 항목으로 통일했습니다.
- `Last Saved` 값은 저장 시각이 있는 프로젝트에서는 실제 일시를, 구버전 데이터처럼 값이 없는 프로젝트에서는 `NoData`를 표시합니다.
- 프로젝트 목록의 상태 표시는 별, 체크, 휴지통 아이콘 기준으로 다시 정렬했고, 카드와 다이얼로그 액션 버튼의 아이콘 스타일도 함께 맞췄습니다.
- 기본 제공 `.slidetheme` 파일에 dark/light 공용 glass 기본값을 추가하고, 구버전 테마 로드 시 자동 fallback 되도록 정리했습니다.
- `glass.noiseOpacity`를 테마 스키마에 추가하고, `Glass Surface`에서 노이즈 강도를 슬라이더와 숫자 입력으로 조절할 수 있게 했습니다.
- 메인 배경의 필름 그레인 레이어는 테마의 `noiseOpacity`를 바로 반영하도록 연결했습니다.
- 에디터 배경에 테마 기반 그라데이션을 추가하고, 헤더·Navigator·슬라이드 카드·편집 패널에 중간 강도의 glass shell을 적용하기 시작했습니다.
- 입력 필드는 완전한 유리층으로 바꾸지 않고, shell 안의 읽기 중심 solid surface로 유지하도록 정리했습니다.
- 테마 모달에 `Glass Surface` 섹션과 live sample card를 추가해 glass 값을 즉시 조정하고 미리볼 수 있게 했습니다.
- glass 색상, alpha, blur, saturation, refraction, depth는 테마 저장 시 `.slidetheme`에 함께 기록되도록 연결했습니다.
- HTML 가이드 export에도 glass 토큰을 연결해, 헤더·Navigator·본문 카드·이미지 wrapper가 에디터와 같은 표면 언어를 공유하도록 맞췄습니다.
- 가이드 본문은 가독성을 우선해 shell 바깥보다 더 짙은 readable surface를 쓰도록 분리했습니다.
- 모바일 폭에서는 header, editor shell, guide shell의 blur/패딩/그림자를 한 단계 낮춰 과한 무게감을 줄이도록 조정했습니다.
- `backdrop-filter` 미지원 환경에서는 glass 표면이 semi-transparent solid fallback으로 내려가도록 기본 폴백을 추가했습니다.
- 라이트모드에서는 헤더 버튼, 가이드 헤더 제목, Navigator 빈 상태 문구와 현재 위치 하이라이트의 대비를 다시 조정해 너무 밝게 날아가던 텍스트와 아이콘을 보강했습니다.
- 좌상단 버전 표시는 pill 배지를 제거하고 더 작은 plain text 메타 정보로 축소했습니다.
- 라이트모드 배경의 좌상단 glow는 고정된 블루 대신 현재 강조색과 가까운 hue를 쓰도록 바꿨습니다.
- 테마 색상에 `secondaryAccent`를 추가하고, hover·soft highlight·보조 강조 glass가 테마별 보조강조색을 따르도록 연결했습니다.
- 다크모드 glass 기본값은 blur와 불투명도를 높이고 highlight/shadow를 낮춰, 라이트모드 대비 과하게 떠 보이던 입체감을 더 플랫하게 정리했습니다.
- 기존 다크 테마가 예전 흰색 glass 기본값을 그대로 저장하고 있더라도, 로드 시 검정 기반의 새 dark glass 기본값으로 자동 보정되도록 마이그레이션 로직을 추가했습니다.
- 메인 배경에는 촘촘한 점선 격자와 약한 노이즈 레이어를 추가해 glass blur가 읽히는 배경 질감을 만들었습니다.
- 라이트모드에서도 헤더 버튼과 프로젝트 표시 pill에 glass 배경이 유지되도록 다시 보정했습니다.
- 새 슬라이드/수정 패널의 이미지 드롭존은 한 덩어리 glass 영역으로 정리하고, hover·dragover 시 tint가 진해지면서 살짝 확대되는 반응을 추가했습니다.
- 이미지가 없는 상태에서는 `스크린샷 이미지 업로드 또는 드래그 앤 드롭` 문구만, 이미지가 있으면 파일명만 보이도록 드롭존 상태를 단순화했습니다.
- 이미지 설명 입력란은 새 슬라이드와 수정 패널 모두 드롭존 바로 아래에만 렌더링되도록 통일했습니다.
- 슬라이드 수정 시 기존 이미지 삭제 체크박스를 제거하고, 새 이미지를 선택한 뒤 저장하면 기존 이미지를 자동으로 교체하도록 기본 동작을 바꿨습니다.
- 미디어 업로드 패널은 본문 입력 옆 가로 레이아웃으로 재배치하고, 전체 패널을 강조색 유리 재질의 드래그 앤 드롭 영역으로 통합했습니다.
- 이미지 설명과 비율 설정은 이미지가 실제로 선택되었을 때만 나타나도록 정리해, 새 슬라이드 입력 단계의 시각적 밀도를 줄였습니다.
- 새 슬라이드/수정 패널의 이미지 드롭존은 별도 탐색 버튼 없이 `스크린샷 이미지 업로드 또는 드래그 앤 드롭` 상태와 업로드 후 파일명 상태만 보여주도록 단순화했습니다.
- 이미지 비율 슬라이더는 본문/미디어 2열 아래의 전체 폭 패널로 옮기고, 테마 강조색을 따르는 range accent로 다시 연결했습니다.
- 새 슬라이드와 수정 패널의 제목, placeholder, 안내 문구를 정상 한글로 정리해 깨진 문자열이 보이지 않도록 다듬었습니다.
- 헤더 우측 프로젝트 표시, 헤더 버튼, 추가/저장/수정/삭제 버튼, 프로젝트·테마 dialog 내부 카드와 정보 행까지 glass shell을 확장했습니다.
- 라이트모드 dialog 안의 입력창과 숫자 필드는 밝은 유리 배경, 보더, inner shadow를 더해 일반 텍스트와 확실히 구분되도록 보정했습니다.
- 페이지, Navigator, 테마/프로젝트 다이얼로그, 코드 블록 등 주요 스크롤 영역의 track/thumb에 같은 glass 스크롤바 스타일을 적용해 통일감을 맞췄습니다.
- 테마 편집기의 `Secondary Accent`와 `Glass Surface` 하위 옵션 라벨을 한글로 정리하고, 닫기 버튼의 `X` 아이콘 정렬도 원형 버튼 중심에 맞춰 보정했습니다.
- Theme, Branding, Project, New Project dialog의 닫기 버튼은 슬라이드 작성 창과 같은 `.btn-cancel` 구조로 통일하고, 헤더 아이콘 여백 규칙을 분리해 X 아이콘이 정확히 가운데 오도록 맞췄습니다.
- 코드블록은 다크 유리 표면을 유지하고, 코드 헤더와 복사 버튼은 라이트/다크 모드에 맞춰 별도 glass 대비를 적용했습니다.
- dialog는 주변 암전/블러 백드롭을 제거하고, 대신 패널 그림자를 강화해 떠 있는 레이어감만 유지하도록 조정했습니다.
- 중제목 전환용 섹션 커버는 라이트모드에서 어두운 유리, 다크모드에서 밝은 유리 표면을 사용하도록 바꿨습니다.
- `--glass-rgb` 값은 쉼표 기반 RGB 문자열로 직렬화해 glass 배경 계산이 안정적으로 동작하도록 보정했습니다.
- 이미지와 설명만 있는 슬라이드도 내부 스크롤바 없이 한 장 안에서 정리되도록 `image-box` 레이아웃과 overflow 처리를 다듬었습니다.
- 프로젝트 저장 시각을 `lastSavedAt`으로 메타와 payload에 함께 기록하고, 프로젝트 매니저의 `Last Saved` 항목이 실제 저장 뒤 시각 문자열로 바뀌도록 연결했습니다.
- 구버전 데이터처럼 저장 시각이 없는 프로젝트는 계속 `NoData`로 표시되도록 fallback을 유지했습니다.

## Fixed

- JSON Import 후 프로젝트 매니저를 불필요하게 닫아 버리던 흐름을 정리해, 현재 모달 문맥을 유지한 채 새 프로젝트 상태를 다시 렌더링하도록 보완했습니다.
- 같은 소제목으로 이어지는 여러 페이지에서도 Navigator 항목은 하나만 유지하면서, 해당 구간 전체에서 같은 항목의 강조가 유지되도록 active 동기화를 수정했습니다.
- 다음 제목으로 넘어갈 때 화면에 조금 남아 있는 이전 슬라이드가 우선 선택되던 문제를 visible area 기준 계산으로 보정했습니다.
