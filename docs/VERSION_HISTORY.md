# Version History Moved

기존 단일 문서형 버전 이력은 hybrid changelog 구조로 이관되었습니다.

앞으로는 아래 파일들을 사용합니다.

- changelog entry point: `CHANGELOG.md`
- unreleased working file: `docs/changelog/unreleased.md`
- release notes folder: `docs/changelog/releases/`
- legacy snapshot archive: `docs/changelog/archive/VERSION_HISTORY_legacy_snapshot.md`

이 파일은 기존 링크 호환을 위한 포인터 문서로만 유지합니다.
