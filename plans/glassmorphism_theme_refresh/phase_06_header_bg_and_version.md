# Glassmorphism Phase 06

## 목표

헤더 상단의 시각적 밀도를 정리하고, 라이트모드에서 배경과 헤더 강조색이 더 자연스럽게 연결되도록 조정합니다.

## 대상 파일

- `src/styles/base.css`
- `src/styles/layout.css`
- `src/features/theme.js`
- `docs/slide_editor_architecture.md`
- `docs/changelog/unreleased.md`

## 구현 범위

### 1. 버전 표시 축소

- 좌상단 `#app-version`은 border/pill 배지를 제거합니다.
- 더 작은 크기의 plain text 메타 정보처럼 보이도록 변경합니다.
- 로고와의 간격은 유지하되, 정보 우선순위는 낮춥니다.

### 2. 라이트모드 좌상단 그라디언트 재매핑

- 현재 라이트모드 배경의 블루 계열 라디얼 글로우를 제거합니다.
- 대신 활성 테마의 강조색과 가까운 hue를 사용하되, 너무 직접적인 원색이 되지 않도록 약간의 hue shift를 둡니다.
- 목표는 "강조색과 맞닿아 있지만 배경으로서 부드러운 톤"입니다.

### 3. 헤더 glass 보강

- 헤더 우측 프로젝트명 pill과 버튼들은 glass shell을 유지합니다.
- 버튼의 fill은 과하지 않게 두되, variant별 강조색은 유지합니다.
- 라이트모드에서도 텍스트 대비가 충분하도록 기본 text tone을 별도 조정합니다.

## 검증 포인트

- 다크/라이트 전환 시 버전 문자열이 항상 읽히는지
- 라이트모드에서 헤더와 배경의 색감이 충돌하지 않는지
- 프로젝트명과 버튼이 glass shell 안에서도 묻히지 않는지

## 완료 기준

- 버전 표시는 pill 없이 작은 텍스트로만 보입니다.
- 라이트모드 좌상단 배경 glow는 강조색 계열로 바뀝니다.
- 헤더 버튼과 프로젝트명은 다크/라이트 모두 읽을 수 있습니다.
